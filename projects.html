Recipe Finder (GitHub)

Recipe Finder

The recipe finder takes a list of ingredients that the user might have at home, and generates a list of recipes that they can create with them. There is a page that returns recipes for food, as well as a page that allows the user to enter cocktail ingredients that they have on hand and returns a list of potential drinks that they can mix.

Technologies Used:

JavaScript, jQuery, HTML, CSS, Materialize, AJAX
